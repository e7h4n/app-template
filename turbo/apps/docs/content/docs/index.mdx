# Welcome to Documentation

This is your project's documentation powered by Fumadocs.

## Features

Fumadocs provides:

- **MDX Support** - Write documentation with React components
- **Full-text Search** - Built-in search functionality
- **Beautiful UI** - Modern and responsive design
- **Dark Mode** - Automatic theme switching
- **Code Highlighting** - Syntax highlighting for code blocks

## Quick Start

1. Edit content in the `/content/docs` directory
2. Use MDX format for your documentation
3. The navigation is automatically generated

## Example Code Block

```typescript
function greet(name: string): string {
  return `Hello, ${name}!`;
}

console.log(greet("World"));
```
